מסרטוני הקלט:
1. VideoToImage.py- לוקח את הסרטונים שבתקיית הקלט ומפרק ושומר את כל פריימים הסרטונים.
2.  בתקיית label Yolo ,Label->ללחוץ על open file ולבחור את תיקיית התמונות, בחירת קובץ טקסט בו כל שורה היא ה label האופציונלים.
  בעזרת תוכנה זו ניתן לסמן בכל תמונה את האובייקטים ונוצר קובץ פלט עם מיקומי האובייטים בתמונה.
3. CreateDataset3.py- מקבל תיקיה עם כל תמונות הפריימים וקיבצי המיקומים, יוצר תיקיה שבה 3 תקיות: א. train ב. valid ג. test ומחלקת את התמונות.
4. train.py-מקבל את תיקית ה train,valid,test עם התמונות וה laybel שלהם וקובץ data מפורמט מתאים, והקוד מאמן את המודל ושומר אותו בתיקיה ובוחן את המודל.
5. toolbarFinal.py,System4-Present.py-(זהים עד כדי הפיצרים שיש בתוכם) נותנים להם את המודל וסרטון, והוא מפעיל את המודל על הסרטון.
